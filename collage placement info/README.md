# ✨ Overview

The SGSITS Placement Information Portal is a modern web application designed to centralize and simplify campus placement information for students.

Instead of searching through multiple notices and PDFs, students can instantly access:

- Company Information
- Salary Packages
- Hiring Locations
- Branch Eligibility
- Recruitment Dates
- Placement Statistics
- Opportunity Tracking

through a clean and responsive dashboard.

# 🚀 Features

## 🔍 Smart Search

Instantly search placement opportunities using:

- Company Name
- Location
- Designation
- Keywords

Students can quickly discover relevant opportunities without manually browsing long lists.

---

## 🎯 Branch-Wise Filtering

Filter opportunities according to branch eligibility.

Supported branches include:

| Branch |
|----------|
| CE |
| CS |
| EC |
| EE |
| EI |
| IP |
| IT |
| MCA |
| ME |
| MT |

---

## 💼 Placement Database

Comprehensive placement records including:

- Company Name
- Package Offered
- Location
- Recruitment Date
- Eligible Branches
- Opportunity Type

---

## 📊 Placement Analytics

The platform provides placement insights such as:

- Total Companies
- Highest Package
- Average Package
- Branch Coverage
- Recruitment Trends

allowing students to better understand placement opportunities.

---

## 💰 Package Tracking

Students can quickly compare opportunities through package information displayed directly on company cards.

Examples include:

- ₹5 LPA
- ₹7 LPA
- ₹8 LPA
- ₹13.4 LPA
- ₹16 LPA
- ₹50 LPA+
