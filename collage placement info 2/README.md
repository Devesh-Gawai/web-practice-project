# ✨ Overview

The SGSITS Campus Placement Portal is a modern web-based dashboard designed to organize and visualize campus placement opportunities in a clean, searchable, and student-friendly format.

Instead of browsing scattered notices and PDFs, students can quickly discover:

- Company Information
- CGPA Eligibility
- Recruitment Categories
- Sector Classification
- Placement Statistics
- Hiring Trends
- Recruitment Timeline

All from a single interactive dashboard.
