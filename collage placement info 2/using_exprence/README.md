## 📂 What's in this Folder?

This directory is organized to document the visual evolution and functionality of the portal. It serves as a portfolio of the user interface and interactive elements.

* **`/Screenshots`** - Static high-resolution images of the dashboard, company grids, and light/dark modes.
* **`/Recordings`** - Screen recordings (like `using_web.mp4`) showing active feature usage, filtering, and animations.
* **`/Assets`** - Logos, custom icons, and chart graphics used within the UI.

---

## ✨ Highlighted Features & Visuals

The media captured in this folder highlights the core functionalities of the platform. Here is a breakdown of the key interfaces documented:

### 1️⃣ Advanced Analytics Dashboard
The hero section and overview dashboard provide real-time metrics for students and placement coordinators.
* **Visualized Data:** Companies by Sector, Company Categories (Super Dream, Dream, Day-01), and CGPA Requirement Spread.
* **Live Counters:** Total Companies, Sectors, and Minimum CGPA thresholds.

### 2️⃣ Dynamic Company Grid & Filtering
The core browsing experience. Videos and images showcase the seamless transition between Grid and List views.
* **Instant Search & Sort:** Filtering by Sector, CGPA requirements, and sorting by Date/Name.
* **Categorized Tags:** Distinct visual badges for *Super Dream*, *Hackathon*, *Off-Campus*, and *Day-01* opportunities.

### 3️⃣ Adaptive UI (Dark & Light Mode)
A modern necessity. Media files demonstrate the seamless toggle between the deep, immersive Dark Mode and the clean, high-contrast Light Mode.
